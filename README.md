# 家計簿（支出管理）アプリ — home-budget
シンプルな支出の登録・一覧・編集ができる Web アプリです。学習・ポートフォリオ用に Laravel で実装しました。
## デモ URL
- **公開 URL:** （ここにデプロイ先 URL を記載）
- 例: `https://example.com`
## 使用技術
- PHP 8.3+
- Laravel 13
- MySQL
- Blade
- Laravel Sail（ローカル開発時の Docker 環境）
- Bootstrap 系ページネーション（Paginator）
## 主な機能
- 支出一覧表示（日付降順、ページネーション）
- 支出の新規登録（日付・カテゴリ・金額）
- 支出の編集・更新
- カテゴリはマスタ（`categories`）から選択
- フラッシュメッセージによる操作結果の表示
## 画面・ルーティング（概要）
| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/` | 一覧・追加フォーム |
| POST | `/post` | 支出の登録 |
| GET | `/edit/{id}` | 編集フォーム |
| PUT | `/update/{id}` | 支出の更新 |
